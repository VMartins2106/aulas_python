
COMANDOS PARA INICIALIZAR ALGO NO GIT E SUBIR PARA O GITHUB:

- ir a pasta desejada
- botão direito na area da pasta -> 'Git Bash Here'

-> git init

-> git add arquivo.py -> adiciona o arquivo ao tree
   	   *.py       -> adiciona todos os arquivos .py
	   .          -> adiciona tudo naquela pasta

-> git status (opicional) -> vê a situação dos arquivos

-> git commit -m "mensagem_desejada" -> finaliza os arquivos

-> git branch -M main -> renomeia a branch de master para main

-> git remote add origin link_repositorio
(origin é um apelido) (link do repositorio criado no github)

-> git push -u origin main -> sobe tudo que foi comitado ao link

----------------------------------------------------------------

CRIANDO NOVAS BRANCHS, PARA NÃO MODIFICAR A ESTRUTURA DA MAIN:

-> git checkout -b 'nova_branch' -> cria outro ramo de desenvolvimento

prepara o add e o commit e usa

-> git push origin nova_branch -> assim insere na nova branch
